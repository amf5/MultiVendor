"# MultiVendor" 
MultiVendor E-Commerce System
A multi-vendor e-commerce system built with Java Spring Boot, featuring full management for admins, vendors, and customers. Includes product management, shopping cart, orders, coupons, payments, and authentication.

Features
Authentication
Customer registration

Login using OTP (One-Time Password)

JWT token-based authentication

Products & Cart
Add products to cart

Update quantity or remove items

Apply or remove discount coupons

Coupons
Create coupons (admin only)

Apply/remove coupons to cart

Retrieve all available coupons

Orders
Create new orders from cart

View user order history

Cancel orders

Payment
Simulated payment processing

Payment status tracking

Administration
Manage users, products, vendors

View reports and statistics

Tech Stack
Java 17

Spring Boot 3.x

Spring Security (JWT)

Hibernate / JPA

MySQL / Oracle (configurable)

Maven

Getting Started
Prerequisites
Java 17+

Database (MySQL)
